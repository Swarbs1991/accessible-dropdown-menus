Intro

A collection of accessible dropdown menu components designed specifically for LocalGov Drupal sites. This demo provides CSS classes and JavaScript functionality to create navigation menus that are keyboard friendly and screen reader accessible.

Usage

This demo uses some classes specific to the LocalGov Drupal project and focuses on improving accessibility when adding child items to the main navigation as dropdown menus. The pattern keeps top-level items as real links while providing a separate, keyboard-focusable toggle to open a submenu.

Specifically done for desktop and large screens.

Key usage notes:

  • Keep anchors as navigation links so they still lead to section landing pages.
  • Use the chevron toggle to expand/collapse submenus without navigating away.
  • ARIA attributes (aria-expanded, aria-controls, aria-hidden) are managed by the script for assistive technologies.

Features

  • WCAG 2.1 AA friendly dropdown menus
  • Keyboard navigation and focus management
  • Screen reader compatible markup and ARIA states
  • Integrates with LocalGov Drupal CSS conventions
  • Responsive behaviour for small screens
